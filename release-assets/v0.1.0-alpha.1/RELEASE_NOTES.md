# MUSE v0.1.0 Alpha 1

This is the first private alpha build of MUSE.

## Installers

- Windows: `MUSE_0.1.0_x64-setup.exe`
- macOS: `MUSE_0.1.0_x64.dmg`
- Linux Debian/Ubuntu: `MUSE_0.1.0_amd64.deb`
- Linux Fedora/RHEL: `MUSE-0.1.0-1.x86_64.rpm`

## Notes for testers

- MUSE is still in alpha. Please keep backups of important writing.
- Windows and macOS may show warnings if the builds are not yet code-signed.
- Please report installation problems and first-launch issues through the alpha feedback link.

## Known issues

No major known blocking issues are currently listed for this build.

Because this is the first alpha release, undiscovered issues are expected. Please report
install problems, first-launch errors, crashes, data loss concerns, slowdowns, confusing
behavior, or anything that interrupts normal writing.
