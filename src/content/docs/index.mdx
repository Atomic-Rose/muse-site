## Built for long-form writers

MUSE is a desktop writing app for novelists, screenwriters, and researchers. It pairs
Scrivener-level organizational power with a modern, elegant editor — and stores everything
in a single `.muse` file on your own computer.

<CardGrid stagger>
	<Card title="Organize like Scrivener" icon="list-format">
		A drag-and-drop binder tree, custom metadata, labels and statuses, smart
		collections, and a Scene Board with seven planning views — from corkboard to
		timeline to story structure.
	</Card>
	<Card title="Write without friction" icon="pencil">
		A rich editor built to handle 80,000-word manuscripts without slowing down.
		Wikilinks, focus mode, typewriter scrolling, and live spelling and grammar help.
	</Card>
	<Card title="Revise with confidence" icon="approve-check">
		A complete review workflow: tracked changes, named revision passes, anchored
		comments, and version snapshots you can roll back to at any time.
	</Card>
	<Card title="Export to real formats" icon="document">
		Compile your manuscript to DOCX (with tracked changes), PDF, or Markdown using
		flexible presets and per-document overrides.
	</Card>
	<Card title="Your work stays yours" icon="approve-check">
		Single-file SQLite projects live on your disk. No cloud account, no telemetry,
		no uploading your manuscript. An automatic backup system keeps you safe.
	</Card>
	<Card title="Calm by design" icon="star">
		High-contrast and dyslexia-friendly options, reduced-motion support, 60
		configurable keyboard shortcuts, and a focused, three-panel layout.
	</Card>
</CardGrid>
