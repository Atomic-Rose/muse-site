This page assumes MUSE is already installed. If not, start with
[Install & first launch](/alpha/install/).

## Create your first project

1. When MUSE opens, you'll see the welcome screen.
2. Choose **New Project** and pick a name and a location on your computer for the `.muse`
   file. (You can move this file later like any other file.)
3. Optionally pick a starting template — templates add suggested folders and gentle writing
   prompts you can ignore or delete.
4. MUSE opens your new project in the main three-panel layout.

:::tip
Your whole project lives in that one `.muse` file. To back it up yourself at any time, just
copy the file. MUSE also makes automatic backups for you — see
[Backups & safety](/manual/backups-and-safety/).
:::

## The three-panel layout

MUSE is organized into three columns:

- **The binder (left).** Your project's table of contents — a tree of folders and documents
  you can drag to reorder. This is where you navigate your manuscript.
- **The editor (center).** Where you write. It can show a single document, several documents
  stitched together (Scrivenings), the Scene Board, or the Outliner.
- **The inspector (right).** Context for the current document: synopsis, notes, metadata,
  writing goals, comments, links, snapshots, and bookmarks.

## Write your first words

1. In the binder, select a document (or create one with the **+** button).
2. Click into the editor and start typing.
3. MUSE saves automatically as you write — there is no save button to remember. The status
   bar shows your save state and word count.
